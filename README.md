# termi-talk-server

Servidor para o projeto Termi‑Talk — chat/terminal minimalista em Node.js/TypeScript.

**Visão geral:**

- **Propósito:** Fornecer a API/servidor em tempo real para o cliente Termi‑Talk.
- **Stack:** Node.js, TypeScript.

**Pré-requisitos:**

- Node.js 18+ instalado
- npm (ou pnpm/yarn) disponível

**Instalação:**

1. Instale dependências:

	`npm install`

2. Execute em modo desenvolvimento:

	`npm run dev`

3. (Opcional) Build para produção:

	`npm run build`

	`npm start`

**Variáveis de ambiente:**

- `PORT` — porta em que o servidor irá escutar (padrão: 3000)
- Outras variáveis específicas (ex.: `NODE_ENV`) podem ser necessárias dependendo da configuração do projeto.

**Uso:**

- Após iniciar com `npm run dev`, o servidor ficará disponível em `http://localhost:PORT`.
- Consulte os endpoints e a lógica de comunicação no código fonte para integrar o cliente.
